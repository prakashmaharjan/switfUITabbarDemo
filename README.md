# SwiftUI Tab Bar Demo

A simple SwiftUI demo showing how to build a basic tab bar interface using `TabView`.

## Overview

This project demonstrates a clean and minimal tab-based layout in SwiftUI with multiple screens:
- Home
- Explore
- Notifications
- Profile

Each tab is represented using system icons and simple placeholder views.

## Features

- SwiftUI `TabView` implementation
- SF Symbols for tab icons
- Multiple independent views per tab
- Custom accent color (green)
- Light color scheme

## Screens

- Home
- Explore
- Notifications
- Profile

## Getting Started

1. Open the project in Xcode
2. Run the app on the simulator or device
3. Tap through tabs to switch between screens

## Tech Stack

- SwiftUI
- iOS 13+

## Purpose

This project is intended as a beginner-friendly example of how to structure and use a tab bar in SwiftUI applications.

Run the app in Xcode to see the tab navigation in action.
